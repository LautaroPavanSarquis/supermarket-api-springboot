# 🛒 Supermarket API – Spring Boot

REST API developed as a technical practice project to simulate a supermarket management system.

## 🚀 Goals

- Practice design patterns
- Build RESTful services
- Test endpoints with Postman
- Dockerize the application

## 🧱 Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Docker
- Postman
